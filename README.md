# Crop Price Forecasting using Machine Learning

This project predicts agricultural crop prices using multiple forecasting models.

Models Used

1 Linear Regression
2 Random Forest
3 Support Vector Regression
4 Decision Tree
5 XGBoost
6 ARIMA

Dataset

Crop Data Monthly 2000–2026

Run Project

pip install -r requirements.txt

python forecast_models.py

Output

results/model_comparison.csv
